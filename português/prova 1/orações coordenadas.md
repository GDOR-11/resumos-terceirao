| Oração coord. | estrutura        |
| ------------- | ---------------- |
| aditiva       | A e B            |
| adversativa   | A, porém B       |
| alternativa   | ou A, ou B       |
| conclusiva    | A, portanto B    |
| explicativa   | A, uma vez que B |
